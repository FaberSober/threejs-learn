## 加载glb、gtlf模型
[https://github.com/pmndrs/](https://github.com/pmndrs/gltfjsx)

1. 使用命令生成加载jsx代码
```bash
npx gltfjsx monkey.glb
```
or
```bash
npx gltfjsx simple-factory.gltf
```
1. 会在当前目录下生成加载jsx代码文件
